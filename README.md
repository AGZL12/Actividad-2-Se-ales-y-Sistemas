# Actividad-2-Se-ales-y-Sistemas
Código hecho en Python utilizando las librerías NumPy y Matplotlib para la generación, graficación y visualización de señales como pulso rectangular, escalón y senoidal.

Se aplica la Transformada Rápida de Fourier (FFT) para analizar las señales en el dominio de la frecuencia y observar su espectro frecuencial.
